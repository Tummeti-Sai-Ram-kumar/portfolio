# Portfolio Website

A simple personal portfolio website built with **React**, **TypeScript**, and **Vite**. It highlights key sections such as introduction, journey, skills, milestones, and contact details.

## Live Demo

🔗 [View the portfolio](https://tummeti-sai-ram-kumar.github.io/portfolio/)

## Tech Stack

- React
- TypeScript
- Vite
- CSS Modules
- ESLint

## Features

- Clean and modern portfolio layout
- Dedicated sections for:
  - Introduction
  - Journey
  - Skills
  - Milestones
  - Contact
- Component-based structure
- Ready for GitHub Pages deployment

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Start the development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview the production build

```bash
npm run preview
```

## Deployment

To deploy the project to GitHub Pages:

```bash
npm run deploy
```
